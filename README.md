# Claude Prompts

A collection of organized prompts and configurations for Claude AI interactions.

## Overview

This repository contains structured prompts, context templates, and skill definitions designed to enhance and customize Claude AI workflows. The repository is organized into three main directories for easy navigation and management.

## Directory Structure

### `/CONTEXT`
Context files that provide background information, system instructions, and environmental setup for Claude interactions. These files help establish the tone, scope, and specific requirements for various AI tasks.

### `/PROMPTS`
A collection of ready-to-use prompts for various use cases and scenarios. These prompts can be used directly or customized for your specific needs.

### `/SKILLS`
Skill definitions and specialized prompt configurations that extend Claude's capabilities in specific domains. These skills can be loaded and chained together to create complex AI workflows.

## Features

- **Organized Architecture**: Clearly separated contexts, prompts, and skills for modularity
- **Reusable Components**: Build and share custom prompts and configurations
- **Flexible Integration**: Easy to integrate into various workflows and applications
- **Comprehensive Coverage**: Contexts, prompts, and skills for diverse use cases

## Getting Started

1. Clone this repository:
   ```bash
   git clone https://github.com/vizi0uz/claude-prompts.git
   cd claude-prompts
   ```

2. Explore the available contexts in `/CONTEXT`
3. Browse ready-made prompts in `/PROMPTS`
4. Review skill definitions in `/SKILLS`
5. Customize and adapt the components to your needs

## Usage

You can use these resources by:
- Loading context files as system prompts
- Using pre-built prompts from `/PROMPTS` directly
- Combining multiple skills for complex tasks
- Adapting the templates to your specific use case
- Chaining contexts, prompts, and skills together for advanced workflows
